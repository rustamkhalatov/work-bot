# Work Bot 🤖

Телеграм-бот для поиска вакансий грузчиков и сборщиков.

## Возможности:
- Просмотр актуальных вакансий по городам.
- Отклик на вакансии через отправку телефона.
- Добавление новых вакансий работодателем.
- Удаление вакансий.

## Готовые города:
- Ростов-на-Дону
- Краснодар
- Волгоград
- Новороссийск
- Пятигорск
